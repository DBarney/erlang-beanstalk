An Erlang beanstalkd client.

Quick start:

  $ make
  ...
  $ erl -pa ebin
  ...
  1> {ok, Q} = beanstalk:connect().
  ...
  2> {inserted, ID} = beanstalk:put(Q, <<"hello world">>).
  ...
  3> {found, ID, Data} = beanstalk:peek(Q, ID).
  ...
  4> {ok, Stats} = beanstalk:stats_job(Q, ID).
  ...
  5> {deleted} = beanstalk:delete(Q, ID).
  ...
  6> ok = beanstalk:close(Q).
  ...
